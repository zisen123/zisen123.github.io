RTX 5090 挖 Pearl (PRL) 加密货币全流程记录
从零开始在 Windows 上通过 WSL2 运行 Pearlhash 矿工，用 RTX 5090 挖 Proof-of-Useful-Work (PoUW) 代币 PRL 的完整折腾记录，包含矿池选择、收益分析、钱包同步等实战经验。
2026-05-30 19:00 label crypto   mining   pearl   prl   rtx5090   wsl2   gpu  

本笔记记录了在 Windows 上使用 RTX 5090 显卡，通过 WSL2 运行 Pearlhash 矿工挖 PRL（Pearl）的完整流程。

1. Pearl 是什么

Pearl 是一个基于 Proof-of-Useful-Work (PoUW) 的 L1 区块链。与比特币的 SHA-256 哈希不同，Pearl 的 PoW 算法是 INT8 矩阵乘法 + BLAKE3 哈希——也就是 GPU 最擅长的事情。挖矿过程中 GPU 执行实际的矩阵乘法运算，作为副产品产出区块证明。

  • 白皮书：https://pearlresearch.ai
  • GitHub：https://github.com/pearl-research-labs/pearl
  • 浏览器：https://explorer.pearlresearch.ai
  • 交易所：https://safetrade.com（PRL/USDT）
  • 币价：约 $1.08（2026-05-30）
参数 数值
算法 pearl (INT8 MatMul + BLAKE3)
区块时间 ~112.8 秒
区块奖励 2688 PRL
全网算力 ~18.24 Eh/s

2. 两条挖矿路径对比

官方 vLLM 矿工 Pearlhash 矿池矿工
原理 跑 Llama-70B 推理 → 矩阵乘法副产品挖矿 直接做 MatMul+噪声+哈希，纯算力挖矿
依赖 Go + Rust + Python + CUDA + C 编译器 一个二进制文件
全节点 需要同步 pearld 不需要
AI 模型 需下载 ~140GB Llama-70B 不需要
5090 兼容 sm90 only (H100/H200) 实测支持 sm120 ✅
复杂度 极高 低

结论：走 Pearlhash 矿池路线。

3. 环境准备

3.1 下载钱包

从 GitHub Releases 下载：

  • Pearl-Wallet-Setup-1.0.0.exe（109MB，图形界面钱包）

安装后创建钱包，记录助记词到纸上，获取 Taproot 格式的收款地址（以 prl1 开头）。

3.2 下载矿工

Pearlhash 矿池提供独立矿工程序：

curl https://pearlhash.xyz/downloads/pearl-miner-v7 -o pearl-miner
chmod +x pearl-miner

矿工是 Linux ELF 二进制（64-bit, x86-64, dynamically linked），在 Windows 上需要通过 WSL2 运行。

3.3 WSL2 确认

检查 WSL2 是否已安装：

wsl --list --verbose

如果未安装：

wsl --install

3.4 Windows Defender 排除

Windows 安全中心 → 病毒和威胁防护 → 排除项 → 添加文件夹 C:\Users\<user>\Downloads\pearl

4. 启动挖矿

4.1 找到矿池连接参数

从 Pearlhash 网页 HTML 中提取的命令格式：

./pearl-miner --host 84.32.220.219:9000 --user <地址>

4.2 一键启动脚本

创建 start-miner.bat（双击启动）和 stop-miner.bat（双击停止）。

启动 PowerShell 脚本核心：

$cmd = "cp /mnt/c/Users/<YourUsername>/Downloads/pearl/pearl-miner-v7.exe /tmp/pm-win 2>/dev/null; chmod +x /tmp/pm-win; /tmp/pm-win --host 84.32.220.219:9000 --user prl1你的地址"

Start-Process wsl.exe -ArgumentList "bash -c `"$cmd`"" -WindowStyle Minimized -PassThru

停止脚本：

wsl bash -c "pkill -9 -f pearl-miner 2>/dev/null"
wsl --shutdown

4.3 启动验证

运行后矿工输出：

GPU #0 (sm: 120) initialized     ← RTX 5090 Blackwell 架构识别成功
Connected to 84.32.220.219:9000
Login success!
Hashrate GPU #0 = 208.29 TH/s

5. 收益分析

5.1 算力数据

配置 算力 功耗
默认频率 ~208 TH/s ~510W
锁功耗（350-400W） ~170-180 TH/s 低噪音
超频（Core +200, PL 575W） ~309 TH/s 590W

5.2 收益计算

  • PRL 价格：$1.08
  • 每天收入（208 TH/s）：~$29
  • 每小时收入：~$1.22
  • 每小时净利（电费 $0.06/kWh）：~$1.17
  • 最低支付门槛：1 PRL
  • 攒够门槛需时：~15-20 分钟

6. PPLNS 矿池机制

6.1 关键概念

概念 说明
PPLNS Pay Per Last N Shares，按最后 N 个份额分配奖励
Immature（未成熟） 区块奖励需等 100 个区块确认后才能使用
最低支付门槛 1 PRL，满额后矿池自动打款

6.2 断开重连

  • 断开立即重连 → 不浪费，份额还在 PPLNS 窗口内
  • 断开很久再回来 → 老份额被挤出窗口，前面白挖

6.3 “网页显示了但钱包为 0”

矿池网页显示的是内部记账，不等于链上转账。等余额成熟（100 确认）且满 1 PRL 后，矿池才会发链上交易。

验证方式：去区块链浏览器搜地址 https://explorer.pearlresearch.ai

7. 关于换地址

  • 每个地址的余额独立计算
  • 切到新地址后，旧地址的余额保留但不会自动合并
  • 旧地址余额不满 1 PRL 会卡住，需切回去继续挖到满
  • Pearl 钱包是 HD 钱包，每次点"收款"生成新地址，但都属于同一个助记词

8. 噪音控制

RTX 5090 满载 510W 风扇非常吵。锁功耗方案：

  • 用 MSI Afterburner 把 PowerLimit 降到 350-400W
  • 算力降到 ~170-180 TH/s 但噪音大幅降低
  • 适合夜间或日常办公场景

9. Apple Silicon (M1/M2/M3/M4) 能挖吗？

理论可行但实践中做不到：

  1. Pearlhash 矿工二进制是 x86-64，且依赖 CUDA
  2. 二进制中包含 mps（Metal Performance Shaders）字符串，说明代码预留了苹果 GPU 路径，但未激活
  3. M1 矩阵乘法算力约相当于 RTX 2060，估计算力 20-50 TH/s，日收入 $3-5
  4. 目前没人成功验证过，需要 ARM64 重新编译

10. 变现路径

PRL 目前唯一上线交易所：SafeTrade（https://safetrade.com）

流程：

  1. 注册 SafeTrade
  2. Pearl 钱包 → SafeTrade 的 PRL 充值地址
  3. 卖 PRL → USDT
  4. USDT 提现到主流交易所 → 换成法币

12. 知识点速查

问题 答案
sm:120 是什么 Blackwell 架构（RTX 5090）的 CUDA 计算能力
Immature txs 等待 100 区块确认的奖励交易
出块固定吗 是，~112.8 秒一个块
中断后重连浪费吗 立刻重连不浪费，隔几小时回来会浪费
换地址余额丢了 不会丢，但旧地址不满 1 PRL 不会打款
